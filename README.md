### Vue

##### 创建 Vue 项目 (vue-base)

```
npm init vue@latest
```

##### 项目目录介绍

```
.vscode            --- VSCode 工具的配置文件
node_modules       --- Vue 项目的运行依赖文件夹
public             --- 资源文件夹（图标浏览器）
src                --- 源码文件夹
.gitignore         --- git 忽略文件
index.html         --- 入口 HTML 文件
package.json       --- 信息描述文件
README.md          --- 注释文件
vite.config.js     --- Vue 配置文件
```
